# Doctor Syria Platform

منصة Doctor Syria هي نظام متكامل لإدارة الخدمات الصحية في سوريا. تربط المنصة بين المرضى والأطباء والصيدليات والمختبرات وشركات الأدوية.

## المميزات الرئيسية

- نظام حجز المواعيد الطبية
- الاستشارات الطبية عن بعد
- إدارة الملفات الطبية الإلكترونية
- نظام الوصفات الطبية الإلكترونية
- خدمة توصيل الأدوية
- نظام إدارة المختبرات
- نظام إدارة الصيدليات
- نظام إدارة شركات الأدوية

## المتطلبات التقنية

- Python 3.9+
- Django 4.2.0
- PostgreSQL
- Redis
- Node.js 16+
- React 18+

## التثبيت

1. قم بإنشاء بيئة Python افتراضية:
```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
venv\Scripts\activate     # على Windows
```

2. قم بتثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

3. قم بإعداد قاعدة البيانات:
```bash
python manage.py migrate
```

4. قم بتشغيل الخادم:
```bash
python manage.py runserver
```
