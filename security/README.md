# نظام الأمان في Doctor Syria

## نظرة عامة
يوفر هذا النظام مجموعة شاملة من الميزات الأمنية لحماية منصة Doctor Syria. تم تصميم النظام وفقاً لأفضل الممارسات الأمنية وتوصيات OWASP.

## المكونات الرئيسية

### 1. أمان كلمات المرور
- التحقق من قوة كلمات المرور
- منع كلمات المرور الضعيفة والشائعة
- تاريخ كلمات المرور السابقة
- تقييم قوة كلمة المرور وتقديم اقتراحات

### 2. أمان الملفات المرفوعة
- فحص الفيروسات
- التحقق من نوع الملف
- تنظيف أسماء الملفات
- حدود حجم الملف
- منع المحتوى الضار

### 3. أمان الجلسات
- إدارة الجلسات الآمنة
- تشفير البيانات
- انتهاء صلاحية الجلسة
- حماية من السرقة

### 4. حماية CSRF
- توليد رموز CSRF
- التحقق من الرموز
- حماية النماذج والطلبات

### 5. سياسة أمان المحتوى (CSP)
- منع XSS
- تحديد المصادر الموثوقة
- حماية من التضمين غير المصرح به

## كيفية الاستخدام

### تفعيل الحماية الأمنية
```python
MIDDLEWARE = [
    'security.middleware.SecurityHeadersMiddleware',
    'security.middleware.ContentSecurityPolicyMiddleware',
    # ... باقي الـ middleware
]
```

### التحقق من كلمة المرور
```python
from security.password_validation import validate_password

try:
    validate_password("كلمة_المرور_هنا")
except ValidationError as e:
    print(f"كلمة المرور غير آمنة: {e}")
```

### التعامل مع الملفات المرفوعة
```python
from security.upload_handlers import SecureFileUploadHandler

handler = SecureFileUploadHandler()
try:
    handler.handle_uploaded_file(request.FILES['file'])
except ValidationError as e:
    print(f"خطأ في الملف المرفوع: {e}")
```
